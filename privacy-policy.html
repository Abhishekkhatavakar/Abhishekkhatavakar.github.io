Privacy Policy – AK Tools Plugins for Revit

Effective Date: July 15, 2025
Publisher: AK Tools

1. Data Collection and Usage

AK Tools plugins for Autodesk® Revit® are designed to operate locally on your computer.

By default, AK Tools plugins do not collect, store, or transmit any personal information or sensitive user data.

The plugins may access project information required to perform their intended functionality, such as:

  • Revit project metadata (e.g., level names, element IDs, categories)
  • Revit application version
  • Plugin diagnostic information and error logs (only if voluntarily shared by the user for support purposes)

This information is used solely to:

  • Provide the plugin's functionality
  • Improve reliability and performance
  • Diagnose and resolve user-reported issues

No personal, financial, or sensitive information is collected.

2. Data Sharing

AK Tools does not sell, rent, or share user data with third parties.

The plugins do not include:

  • Analytics services
  • Advertising SDKs
  • User tracking
  • Cloud-based data collection

If future versions introduce optional online services or third-party integrations, this Privacy Policy will be updated accordingly.

3. Data Storage

AK Tools plugins process data locally on the user's computer.

If diagnostic logs are voluntarily provided for technical support:

  • They are used only to investigate the reported issue.
  • They are retained only for as long as necessary to resolve the issue.
  • They are deleted when no longer needed.

AK Tools does not automatically upload project files or user information.

4. User Rights

Users may request deletion of any diagnostic information they have voluntarily shared for support purposes.
